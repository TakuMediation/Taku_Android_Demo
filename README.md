# Taku-Android-Demo SDK for Android

## Requirements

We support Android Operating Systems Version 4.1 (API Level 16) and up. Be sure to:

- Use Android Studio 3.0 and up
- Target Android API level 33
- MinSdkVersion level 16 and up
